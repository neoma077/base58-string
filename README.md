# base58-string

## What?

base58 encode decode without buffer;  
input string out string;  
suport browser;

## Installation

```shell
npm install base58-string
```

## Usage

```javascript
const Bs = require('base58-string');
Bs.encode('base58'); // 'qzTiEHgB'
Bs.decode('qzTiEHgB'); // 'base58'
```
